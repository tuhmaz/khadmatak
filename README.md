## 📋 نظرة عامة على المشروع

### الهدف الرئيسي
حل مشكلة العثور على فنيين موثوقين ومهرة للخدمات المنزلية في الأردن من خلال منصة رقمية آمنة وسهلة الاستخدام.

### المميزات الحالية ✅
- **واجهة مستخدم جذابة** باللغة العربية مع دعم RTL كامل
- **8 فئات خدمات رئيسية**: السباكة، الكهرباء، النجارة، التنظيف، التكييف، الدهان، الحدائق، نقل الأثاث
- **نظام مصادقة متكامل** مع JWT وتشفير كلمات المرور
- **تسجيل مستخدمين متقدم** للعملاء ومقدمي الخدمات
- **لوحات تحكم تفاعلية** لكل نوع مستخدم
- **نظام عرض مقدمي الخدمات** مع التقييمات والمراجعات
- **نظام طلب الخدمات محمي** مع التحقق من الهوية
- **عرض الطلبات والإحصائيات** في لوحات التحكم
- **تصميم متجاوب** يعمل على جميع الأجهزة
- **تحميل ديناميكي للبيانات** عبر API محمي
- **نظام إشعارات تفاعلي** للمستخدمين

## 🏗️ المعمارية التقنية

### التقنيات المستخدمة
- **Backend**: Hono Framework مع TypeScript
- **Platform**: Cloudflare Workers/Pages (Edge Computing)
- **Database**: Cloudflare D1 (SQLite المُوزع) - جاهز للتطبيق
- **Frontend**: Vanilla JavaScript مع TailwindCSS
- **Icons**: FontAwesome 6.4.0
- **HTTP Client**: Axios

### قاعدة البيانات 🗄️
**تصميم شامل لقاعدة بيانات متكاملة (جاهز للتطبيق)**:

#### الجداول الرئيسية:
1. **users** - العملاء ومقدمي الخدمات والإدارة
2. **service_categories** - فئات الخدمات (8 فئات حالياً)
3. **service_providers** - ملفات مقدمي الخدمات الشخصية
4. **provider_services** - الخدمات المقدمة من كل مزود
5. **service_requests** - طلبات الخدمة من العملاء
6. **reviews** - التقييمات والمراجعات
7. **messages** - نظام المراسلة
8. **payments** - إدارة المدفوعات والعمولات
9. **notifications** - إشعارات النظام

#### المميزات المتقدمة في قاعدة البيانات:
- **فهرسة محسّنة** للاستعلامات السريعة
- **دعم الموقع الجغرافي** (GPS coordinates)
- **نظام تقييم متطور** مع إمكانية الرد
- **تتبع شامل للطلبات** مع حالات متعددة
- **نظام عمولات مرن** للمنصة

## 🔐 نظام المصادقة والأمان

### مميزات الأمان المُطبقة ✅
- **تشفير كلمات المرور** باستخدام bcrypt مع salt عشوائي
- **JWT آمن** مع توقيع رقمي وانتهاء صالحية (24 ساعة)
- **حماية ضد الهجمات** مع rate limiting للطلبات
- **جلسات آمنة** مع HTTP-only cookies
- **التحقق من البيانات** (إيميل، رقم هاتف أردني، قوة كلمة المرور)
- **حماية من XSS** مع تنظيف المدخلات
- **CORS محدود** للـ API endpoints

### أنواع المستخدمين
1. **العملاء (Customers)**
   - تسجيل بسيط مع البيانات الأساسية
   - إمكانية طلب الخدمات
   - متابعة الطلبات والتقييم

2. **مقدمو الخدمات (Providers)**
   - تسجيل مفصل مع معلومات العمل
   - اختيار فئات الخدمات المتخصص بها
   - حالة التحقق (محقق/قيد المراجعة)
   - لوحة تحكم متقدمة مع الإحصائيات
